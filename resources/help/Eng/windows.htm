The Windows view

On the Windows tab you can monitor windows and applications running.

  • Handle

    • The handle of a window (hex value).

  • Caption

    • Text of the specified window's title bar.

  • Class Name

    • The name of the class to which the specified window belongs.

  • Rectnagle

    • The dimensions of the bounding rectangle of the specified window in the following form:
      (Left,Up,Right,Bottom) Width x Height
      Left,Up - coordinates of the upper-left corner.
      Right,Bottom - coordinates of the lower-right corner.
    The coordinates are relative to the upper-left corner of the screen.

  • Style

    • The window-class style bits.

  • Extended Style

    • The extended style of the window.

  • Status

    • Determines whether the Microsoft Windows considers that a specified application is not responding, or "hung". An application is considered to be not responding if it is not waiting for input and is not in startup processing.
    The value can be one of the following: Running, Not Responding.
