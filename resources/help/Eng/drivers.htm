The Drivers view

On the Drivers tab, you can view information about installed drivers.

  • File

    • The module's filename.

  • Path

    • The full path and filename of the driver.

  • Description

    • File description.

  • Company

    • The company that produced the file.

  • Version

    • File version number.

  • Image Base

    • The base virtual address where the image is loaded (hex value).

  • Image Size

    • The size of the image (hex value).



Context Menu items:
  • File Properties

    • This selection opens a property dialog that shows you information about a file selected.
