The Process Context Menu

  • Priority

    • You can change the base priority of a process with this submenu. When you change the base priority of a process the system adjusts the priorities of threads within the process so that they remain at the same relative priority with respect to the new base priority.

  • Launch Depends

    • If PROWiSe Manager finds the Dependency Walker tool, it will be opened with the selected process as the argument.
    * The Dependency Walker tool shows static DLL dependencies. (see http://www.dependencywalker.com)
  • Suspend

    • If you want a process to become temporarily inactive, so that a system resource such as network, CPU or disk, becomes available for other processes, you can suspend the process. To resume a suspended process chose the Resume item from the process context menu.
    * Suspended processes show in a dark grey color.

  • Kill process

    • This item terminates a process with the Terminate Process API. Note that a process terminated in this way is not warned of its termination and therefore does not write unsaved data it may have.

    Important! Be careful when ending a process. If you end a system service, some part of the system might not function properly.

  • Properties

    • This selection opens a property dialog that shows you more information about a process.
