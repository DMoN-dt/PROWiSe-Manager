System Shutdown

  • Lock

    • Locks the workstation's display, protecting it from unauthorized use while your programs and files remain open, and lets another user log on.
    You can also lock workstation by pressing the Windows logo key + L.

  • Log Off

    • Closes your programs and ends your Windows session.

  • Hibernate

    • Saves your session to disk so that you can safely turn off power. Your session is restored the next time you start Windows.

  • Restart

    • Shuts down Windows and then starts Windows again.

  • Turn Off

    • Ends your session and shuts down Windows so that you can safely turn off power.

  • Stand By

    • Puts your computer in a low-power state so that you can quickly resume your Windows session.

If the operation is unavailable, a corresponding button is grayed.

The following parameters are used at a log-off, reboot and turn off.
  • Normal Windows Exit

    • The shutdown goes as through the Start button. The system forces hung applications to close.

  • Forced Windows Exit

    • The system forces applications to close. Note that all unsaved data will be lost!

  • Turbo Windows Exit

    • PROWiSe Manager calls NtShutdownSystem function. The system shuts down within 5 seconds. All unsaved data will be lost!
    Use only in an emergency.

  • Use Power-Off Feature (at turn off)

    • Select this item if your system supports the power-off feature.
