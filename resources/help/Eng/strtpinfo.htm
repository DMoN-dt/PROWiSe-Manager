The Startup view

On the Startup tab you can see the programs that runs automatically at Windows startup.

  • Item

    • Value of the given field depends on item location. This member can be one of the following values:
    • Name of registry value
    • File name (for files and shortcuts in Startup folder)
    • Explorer BHO (Explorer Browser Helper Object)
    • IExplorer Plugin (for files located in %PROGRAM_FILES_DIR%\Internet Explorer\Plugins )

  • Item Parameters

    • Value of the given field depends on item location. This member can be one of the following values:
    • The data of registry value
    • The path to executable file (for shortcuts in Startup folder)
    • Name of registry sub-key (for Explorer Browser Helper Object)
      • * Browser Helper Object - plugin to Internet Explorer.
    • The file name (for IExplorer Plugin )

  • Item Location

    • Value of the given field depends on item location. This member can be one of the following values:
    • Registry key
    • Path to the shortcut (for shortcuts in Startup folder)
    • Path to the file (for IExplorer Plugin)



Context Menu items:
  • Jump to Location
    • If the Location is registry, PROWiSe Manager will launch the Registry Editor (if it hasn’t been launched already) and navigate directly to the value or key.
    • If the Location is a folder, PROWiSe Manager will open it in Windows Explorer.

  • Jump to Path

    • If the Item Parameters is a valid path, PROWiSe Manager will open it in Windows Explorer.

  • CLSID information

    • PROWiSe Manager opens a dialog box with data of the following values from the registry: Default, ProgID, InprocServer, InprocServer32, LocalServer, LocalServer32.

  • Jump to CLSID

    • PROWiSe Manager will launch the Registry Editor and navigate directly to the sub-key of HKEY_CLASSES_ROOT\CLSID\{...}

  • Delete Item

    • Depending on the item, the registry value will be removed, or the file will be deleted. If the item is the shortcut then it will be deleted, instead of an executable file.
