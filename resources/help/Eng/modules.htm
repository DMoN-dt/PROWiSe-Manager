The Modules pane

On the Modules pane, you can view information about libraries loaded by selected process.

  • File

    • The module's filename.

  • Path

    • The full path and filename of the driver.

  • Description

    • File description.

  • Company

    • The company that produced the file.

  • Version

    • File version number.

  • Image Base

    • The base virtual address where the image is loaded (hex value).

  • Image Size

    • The size of the image (hex value).

  • Image Base Address

    • During creation of executable module, a linker guesses, that the file will be mapped at a specified place in memory. This address is the Image Base Address. If operating system can't map a file at this address, the system's loader makes some code modification.
    * Relocated module show in yellow color.



Context Menu items:
  • File Properties

    • This selection opens a property dialog that shows you information about a file selected.

  • Launch Depends

    • If PROWiSe Manager finds the Dependency Walker tool, it will be started with the selected DLL as a command-line argument.
