General Options

  • Always on Top

    • Choose this option to have PROWiSe Manager's window remain above other windows.

  • Hide When Minimized

    • Select this option to have PROWiSe Manager run in the tray as a small graph reflecting current CPU usage when you minimize it.

  • Hide on ESCape instead of close

    • Select this option to have PROWiSe Manager hide main window instead of exit when you press ESCape button. (takes effect only if it runs with /hide parameter)

  • Allow Only One Instance

    • Check this to prevent multiple instances of PROWiSe Manager to run simultaneously.

  • Replace Task Manager

    • Select this option to have PROWiSe Manager execute instead of Task Manager when you launch Task Manager. Note that this is a global setting that affects all users regardless of how they start Task Manager.

  • Language

    • Select a language for the interface.
