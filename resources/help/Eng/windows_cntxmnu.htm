The Window Context Menu

  • Switch To

    • Brings the selected window to the foreground.

  • Always On Top

    • Choose this option to have this window remain above other windows.

  • Minimize / Maximize

    • Minimizes/maximizes the specified window.

  • End Task

    • Select this option to forcibly close a specified window. In appeared window you can choose between Normal and Force. If you select Force the destruction of the window will be forced if an initial attempt fails to gently close the window.

  • Process Properties

    • This selection opens a property dialog that shows you information about a process created the specified window.
